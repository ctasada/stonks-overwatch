# Application

The Stonks Overwatch application is available for Windows, macOS, and Linux. It is designed to run locally, ensuring that your data remains private and secure.

The application provides a user-friendly interface to access and manage your investments across different trackers. It allows you to view your portfolio, track performance, and analyze your investments in real-time.

## HOW TOs

### Execute in Debug mode

In case you need to debug the application, you can run it in debug mode. This will create extra logs.

**macOS:**

```shell
DEBUG=true open -n /Applications/Stonks\ Overwatch.app/
```
