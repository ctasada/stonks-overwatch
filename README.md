# Stocks Portfolio

## Start Developing

```shell
python3 -m venv venv
source venv/bin/activate
```

For Windows
```shell
python3 -m venv venv
.\venv\Scripts\activate.bat
```

Install Dependencies
```shell
pip install -r requirements.txt
```

Update Dependencies
```shell
pip freeze > requirements.txt
```

Run the server
```shell
python manage.py runserver
```
